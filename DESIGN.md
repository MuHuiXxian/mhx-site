# 穆惠娴个人网站 — 设计初稿 v1

## 设计理念

- **极简至上**：第一版只传达一件事——「这是穆惠娴，她做后端开发，这是她的作品」
- **先存在，再完美**：上线比好看重要。第一版只有白底 + 一种主题色
- **内容是主角**：去掉一切装饰元素，让文字和项目说话

## 配色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 背景 | 白色 | `#FFFFFF` |
| 主题色 | 墨绿 | `#166534`（Tailwind green-800） |
| 文字主色 | 深灰 | `#1F2937`（Tailwind gray-800） |
| 文字副色 | 中灰 | `#6B7280`（Tailwind gray-500） |
| 分割线/边框 | 浅灰 | `#E5E7EB`（Tailwind gray-200） |
| 链接悬停 | 浅墨绿 | `#F0FDF4`（Tailwind green-50） |

**规则：只用这些颜色。** 不引入第三种主题色，不准加渐变色。想换颜色？上线后再改。

## 字体

- 中文/正文：系统默认字体（`system-ui`），不做任何 web font 引入
- 英文/代码：`JetBrains Mono`（你写代码用的字体，保持一致性）
- 字号层次：标题 2xl，副标题 xl，正文 base，小字 sm

## 技术栈

| 层 | 技术 | 理由 |
|-----|------|------|
| 框架 | Next.js 16 (App Router) | 独立开发者首选，SSG 让博客秒开 |
| 样式 | Tailwind CSS v4 | 不用写 CSS 文件，class 一把梭 |
| 组件库 | shadcn/ui | 复制粘贴式使用，不引入整个包 |
| 语言 | TypeScript | 你写 Java 的，类型系统顺手 |
| 内容 | Markdown (.mdx) | 文章直接写 Markdown，不需要数据库 |
| 部署 | Vercel | 免费、Git 推送自动部署、自带域名 |
| 分析（可选） | Vercel Analytics | 看看有多少人访问，不加也行 |

## 第一版不做清单

- ❌ 暗黑模式
- ❌ 评论系统
- ❌ CMS / 数据库
- ❌ 用户登录
- ❌ 动画效果（包括页面过渡、hover 动画）
- ❌ Web font（系统默认字体足够）
- ❌ RSS 订阅
- ❌ SEO 深度优化（有基本 meta 标签就行）
- ❌ 音乐播放器
- ❌ 访问统计
- ❌ 多语言
- ❌ 管理后台
