## Week 1：前端基础 + 首页（6/22 — 6/28）

### 学习目标
- 能写出 React 函数组件，理解 props 和 useState
- 能用 Tailwind 写一个响应式页面
- 首页在本地跑起来，内容完整

| 时间 | 内容 | 方式 |

## Week 2：期末周降级模式（6/29 — 7/5）

### 原则
- **不写新代码，不学新概念**
- 期末优先，网站只做「翻阅」级别的投入
- 如果期末复习累了，允许做以下低认知负荷的事：

### 允许做的事
| 事项 | 耗时 | 说明 |
|------|------|------|
| 改首页文案 | 15min | 换成更满意的自我介绍 |
| 调整颜色 | 10min | 墨绿看腻了就换，但只能一种主题色 |
| 添加文章 Markdown | 30min | 从日记里挑一段改写成文章草稿，存到 `content/blog/` |
| 逛 react.dev 闲读 | 自由 | 翻「Escape Hatches」章节，不用敲代码 |
| 看看别人的个人网站 | 30min | Product Hunt / Indie Hackers 搜个人网站，收集灵感 |

### 禁止做的事
- 不要学新框架（Solid.js / Vue / Svelte / Astro）
- 不要重构代码
- 不要加新功能
- 不要在深夜探索新技术导致失眠

**本周唯一目标：期末不挂科。网站是次要的。**

## 后端手搓能力提升 —— 长期计划

个人网站是第一关。之后按这个节奏持续锻炼：

```
第一阶段（现在）    ：不看视频手搓 CRUD 模块
第二阶段（暑假前半） ：不看视频手搓完整业务（含权限、文件上传、定时任务）
第三阶段（暑假后半） ：自己设计并实现一个新项目（非苍穹外卖）
第四阶段（大二上）   ：学习 Linux + Docker，把项目部署到自己的服务器
```

### 检验标准

能不看视频 / 不看源码，在空白 IDEA 中独立完成：

- [ ] 建表 SQL
- [ ] Entity + Mapper + Service + Controller 三层
- [ ] 参数校验（@Valid）
- [ ] 统一返回格式（Result 类）
- [ ] 全局异常处理（@RestControllerAdvice）
- [ ] 分页查询 + 模糊搜索
- [ ] JWT 登录验证 + 拦截器
- [ ] 文件上传（OSS）

**通过标准：8 项中有 6 项能独立写出来，就算过关。**
